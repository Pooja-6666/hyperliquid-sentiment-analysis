# hyperliquid-sentiment-analysis
Analyzing 211K Hyperliquid trades against the Bitcoin Fear &amp; Greed Index to uncover how market sentiment shapes trader behavior and profitability.
# Trader Behavior & Market Sentiment Analysis

## Overview
Analysis of 211,224 Hyperliquid trades against the Bitcoin Fear & Greed Index (2024).

## Key Findings
- Fear periods generated the highest total PnL ($3.35M)
- Traders are contrarian: 64% Long in Fear, 57% Short in Greed
- Top 20 traders maintain 90%+ win rate regardless of sentiment

## Files
- `trader_sentiment_analysis.ipynb` — full analysis notebook
- `fear_greed_index.csv` — sentiment dataset
- `historical_data.csv` — Hyperliquid trade data

## Tools
Python, pandas, matplotlib, seaborn, scipy
